Fast Inversion of Control Tool and Sundry Items of Roslyn Chicanery


Inversion of Control Container

A fast IoC container heavily optimized for usage within ASP.Net Core and other .Net server side applications. The successor to the venerable StructureMap library, the original IoC tool for .Net.

Dynamic Code Weaving

Unlock an all new world of dynamic code in C# without the headaches of Expression trees or esoteric IL emitting by just generating nice, clean C# in memory and using Roslyn to build .Net types on the fly.
